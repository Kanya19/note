对于s = x + y；如何判断s是否发生过溢出？简单来说就是 判断 s < x 或者 s < y 如果成立则表示发生了溢出。证明如下：假设x + y溢出，则 s = x + y - 2^w ; 由于 [公式] ，所以 y - 2^w < 0，可得 s < x；此对无符号数有效
### 二进制补码加法
    对于数值执行二进制加法，然后直接截断到指定位数，即可得到想要的结果。
    无论有符号无符号，二进制加法都是直接按位相加，超出最大位的就丢掉就对了。
